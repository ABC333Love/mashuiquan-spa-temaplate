# spa-template

##  项目主要技术
    1. 框架： vue大礼包（vue + vue-router + vuex）
    2. UI库： vant
    3. http模块： axios
    4. 打包工具： vue-cli3
    5. 自适应方案： flexible.js + postcss-pxtorem(postcss.config.js配置)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production

npm run build-prod
